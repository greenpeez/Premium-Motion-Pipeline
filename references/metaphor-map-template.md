## Object Rules

For every object, define:

- Name.
- What it represents.
- Whether it is 2D, 2.5D, or 3D.
- Source-of-truth reference.
- Material/color rules.
- Allowed poses.
- Whether it can be reused elsewhere.
- Whether it needs labels. Prefer no labels inside premium product animations unless approved.

## Red Flags

Return to metaphor work if:

- The animation is only moving circles, lines, blobs, or diagrams without clear product meaning.
- The user says it feels abstract.
- Objects do not explain the product relationship.
- The same motion could describe any generic app.

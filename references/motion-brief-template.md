## Required Clarifications

Ask only when a decision changes the visual direction or production path:

- What should the metaphor literally show?
- Which references are source-of-truth?
- Which objects must be accurate?
- Which artifacts are unacceptable?
- Which checkpoint can be autopiloted?

## Translation Rules

- "Premium" -> slow cadence, intentional spacing, stable camera, clean depth, no jitter.
- "Simple" -> fewer visible elements, literal object roles, no explanatory labels inside the animation.
- "Beautiful" -> approved style frame, not just motion.
- "Smooth" -> continuous paths, consistent cadence, no teleporting, correct easing.
- "Interactive scroll" -> pinned section, scroll-scrubbed timeline, reversible states.
- "Loop" -> seamless first/last frame, no visible reset, no abrupt spawn unless masked or faded.
